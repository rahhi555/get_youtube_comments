# Youtubeのコメントを翻訳してスプレッドシートに出力するGoogle Apps Script

ローカルのvscodeで開発し、`clasp`というGAS版のgithub的なものでAppsScriptにプッシュしています。

基本はGoogle翻訳ですが、APIキーを用意いただければDeepL翻訳も使用できます。
